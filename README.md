# 🏦 Calculateur Bancaire — Gestion des Comptes

Application web interactive de calcul financier bancaire, développée en HTML, CSS et JavaScript pur.

---

## 📌 Fonctionnalités

### 💰 Compte d'Épargne
- Calcul des intérêts selon le capital initial, le taux annuel et la durée
- Saisie par dates de début et de fin de placement
- Affichage détaillé des résultats (intérêts, capital final)

### 🏧 Compte Courant
- Gestion des opérations (retraits et versements)
- Calcul automatique des dates de valeur selon le type d'opération
- Génération de l'échelle d'intérêts avec soldes et intérêts bruts par période

### 📄 Bordereau d'Escompte
- Calcul de l'escompte commercial sur effets de commerce
- Prise en charge des frais optionnels : TVA, commission d'endos, commission de service
- Génération du bordereau d'escompte complet

## 🚀 Lancement du projet

Aucune installation requise. Il suffit de cloner le dépôt et d'ouvrir le fichier HTML :

```bash
git clone https://github.com/asmahajji/calculateur-bancaire.git
cd calculateur-bancaire
# Ouvrir index.html dans votre navigateur
```
